1. Перед началом нужно загрузить на Google Диск (https://drive.google.com/drive/) фотографию для изменения.
2. Необходимо перейти на официальный сайт Google Colaboratory (https://colab.research.google.com/) и загрузить файл diploma.ipynb в среду разработки.
3. Выбрать аппаратный ускоритель: T4 GPU.
4. Выбрать тип среды выполнения: Python 3.
3. Раскрыть раздел «Измененние на основе текстовых описание».
4. Загрузить всё из подраздела «Зависимости».
4. Соглашаться со всеми всплывающими окнами.
5. Загрузить всё из подраздела «Исходный код».
6. Раскрыть подраздел «Программа».
7. В блоке с командой !cp указать путь до файла на диске
8. Последовательно запускать все команды 
9. В форме изменения изображения ввести чувствительность, количество шагов и количество изображений.
10. Нажать продолжить.
11. На новой форме выбрать размер кисти. На изображении выделить область которая будет измененина.
11. Написать текст (на английском языке) того что будет нарисовано в выделенной области. 
12. Нажать поехали.
13. Выбрать изображение из дропдауна поехали.
14. Нажать «Продолжить», для продолжения изменения изображения или же «Останость» для окончанияю.
15. Запустить следующий блок кода для вывода результата.

Для разделов «Автоматическая сегментация» и «Сегментация по точкам» последовательность действий аналогичная.
